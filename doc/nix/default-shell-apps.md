`z` - https://github.com/rupa/z
- install (no *brew)
```
git clone https://github.com/rupa/z/ ~/tmp/z
chmod +x ~/tmp/z/z.sh
mv ~/tmp/z/z.sh /usr/local/bin/
mv ~/tmp/z/z.1 /usr/local/share/man/man1
rm -rf ~/tmp/z
```
